发表于 2024-06-03 更新于 2024-06-04 分类于 Algorithm

二分图

一些概念

二分图 : 分左右两部分，各部分之间的点没有边连接

最大匹配：二分图中选出一些边，使得这些边没有公共顶点，且边的数量最大

最小点覆盖：选最少的点，满足每条边至少有一个端点被选。最小点覆盖 = 最大匹配

最大匹配数为n，则有n个条边没有公共顶点。最大匹配中的每条边被覆盖至少需要n个点。
每个点至少会连接一条最大匹配中的边。

最大独立集：选最多的点，满足两两之间没有边相连。最大独立集 = 最小点覆盖

在最小点覆盖中，任意一条边都被至少选了一个顶点，所以对于其点集的补集，任意一条边都被至多选了一个顶点，所以不存在边连接两个点集中的点，且该点集最大。

tarjan 桥与割点

发表于 2024-06-03 更新于 2024-06-04 分类于 Algorithm

tarjan 桥与割点

DFN(u)为节点u 搜索的次序编号(时间戳);

LOW(u)为u 或 u的子树能够追溯到的最早的栈中节点的次序号;

由定义可以得出，当 DFN(u)=LOW(u)时，以u为根的搜索子树上所有节点是一个强连通分量。

桥

在dfs树中，如果v节点的父节点时u，且low[v]>dfn[u]，说明v子树中没有其他路径通往u及祖先，所以(u,v)是桥，移除后会让连通分量增加。

割点

求根节点单独处理，通过子树个数超过2判断。
