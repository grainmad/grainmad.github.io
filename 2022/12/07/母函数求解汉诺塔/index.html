жҜҚеҮҪж•°жұӮи§ЈжұүиҜәеЎ”

еҸ‘иЎЁдәҺ 2022-12-07 жӣҙж–°дәҺ 2022-12-08 еҲҶзұ»дәҺ з»„еҗҲж•°еӯҰ

д»Җд№ҲжҳҜжҜҚеҮҪж•°

еңЁж•°еӯҰдёӯпјҢжҹҗдёӘеәҸеҲ—${\displaystyle (a_{n}){n\in \mathbb {N} }}(a{n})_{n\in \mathbb {N} }$ зҡ„жҜҚеҮҪж•°пјҲеҸҲз§°з”ҹжҲҗеҮҪж•°пјҢиӢұиҜӯпјҡGenerating functionпјүжҳҜдёҖз§ҚеҪўејҸе№Ӯзә§ж•°пјҢе…¶жҜҸдёҖйЎ№зҡ„зі»ж•°еҸҜд»ҘжҸҗдҫӣе…ідәҺиҝҷдёӘеәҸеҲ—зҡ„дҝЎжҒҜгҖӮ

жҜҚеҮҪж•°е°ұжҳҜдёҖеҲ—з”ЁжқҘеұ•зӨәдёҖдёІж•°еӯ—еәҸеҲ—зҡ„жҢӮиЎЈжһ¶гҖӮ

е®ҡд№үпјҡеҜ№дәҺ$c_0,c_1,c_2,\dots$ жһ„йҖ еҮҪж•°$G(x) = c_0+c_1x+c_2x^2+\dots$ з§°$G(x)$дёәеәҸеҲ—$c_0,c_1,c_2,\dots$ зҡ„жҜҚеҮҪж•°

жұӮи§ЈжұүиҜәеЎ”

зҺ°жңүзј–еҸ·$A$гҖҒ$B$е’Ң$C$дёүж №жҹұеӯҗпјҢжңү$n$дёӘдёӯз©әзӣҳеӯҗеңЁ$A$жҹұеӯҗдёҠпјҢзӣҳеӯҗеӨ§е°Ҹдә’дёҚзӣёеҗҢд»Һе°ҸеҲ°еӨ§дҫқж¬ЎжҺ’еҲ—пјҢжҜҸж¬ЎеҸӘиғҪз§»еҠЁжңҖдёҠйқўзҡ„зӣҳеӯҗдё”з§»еҠЁеҗҺиҰҒдҝқиҜҒе°ҸзӣҳеӯҗеңЁеӨ§зӣҳеӯҗдёҠгҖӮ

зҺ°еңЁйңҖиҰҒе°Ҷ$A$жҹұеӯҗдёҠзҡ„$n$дёӘзӣҳеӯҗз§»еҠЁеҲ°$C$дёҠгҖӮеҸҜеҲҶдёүжӯҘиө°пјҢйҰ–е…Ҳз§»еҠЁ$n-1$дёӘзӣҳеҲ°$B$жҹұеӯҗдёҠпјҢз„¶еҗҺе°ҶжңҖеҗҺдёҖдёӘзӣҳеӯҗз§»еҠЁеҲ°$C$дёҠпјҢжңҖеҗҺе°Ҷ$B$дёҠзҡ„$n-1$дёӘзӣҳеӯҗз§»еҠЁеҲ°$C$дёҠгҖӮ

жҲ‘д»¬и®ҫз§»еҠЁ$n$дёӘзӣҳеӯҗеҲ°еҸҰдёҖж №жҹұеӯҗдёҠйңҖиҰҒ$f(n)$жӯҘпјҢж №жҚ®дёүжӯҘиө°жңүйҖ’жҺЁе…¬ејҸ$f(i) = 2 * f(i-1) + 1,f(0) = 0$гҖӮ

жһ„йҖ жҜҚеҮҪж•°$G(x) = f(0) + f(1)x + f(2)x^2 + f(3)x^3 + \cdots$

жҜҚеҮҪж•°еҸҜдҪңдёәйҖ’жҺЁе…¬ејҸе’ҢйҖҡйЎ№е…¬ејҸзҡ„жЎҘжўҒпјҢиҝҷж—¶еҖҷжҲ‘д»¬иҰҒ$f(n)$е°ұеҸӘиҰҒжұӮеҮә$x^n$зҡ„зі»ж•°еҚіеҸҜгҖӮ

е°ҶйҖ’жҺЁе…¬ејҸеёҰе…Ҙ$G(x)$еҸҜеҫ—

$G(x) = (2f(0)+1)x+(2f(1)+1)x^2+(2f(2)+1)x^3+\cdots$

$G(x) = 2x(f(0)+f(1)x+f(2)x^2+\cdots) + x + x^2 + x^3 + \cdots$

$G(x) = 2xG(x) + x+ x^2 + x^3 + \cdots$

ж №жҚ®жі°еӢ’е…¬ејҸ

$G(x) = 2xG(x) + \frac{1}{1-x}-1$

$G(x) = \frac{x}{(1-2x)(1-x)} = \frac{1}{1-2x} - \frac{1}{1-x} = (1+2x+ 2^2x^2 + 2^3x^3 + \cdots)-(1+x+ x^2 + x^3 + \cdots)=\sum \limits_{i=0}^{\infty} (2^i-1)x^i$

so $f(i) = 2^i-1$
