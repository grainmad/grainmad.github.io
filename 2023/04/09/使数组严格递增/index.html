1187. 使数组严格递增

给你两个整数数组 arr1 和 arr2，返回使 arr1 严格递增所需要的最小「操作」数（可能为 0）。

每一步「操作」中，你可以分别从 arr1 和 arr2 中各选出一个索引，分别为 i 和 j，0 <= i < arr1.length 和 0 <= j < arr2.length，然后进行赋值运算 arr1[i] = arr2[j]。

如果无法让 arr1 严格递增，请返回 -1。

方法一：

思路

我们令dp[i][j]为将第i个数改为arr2[j]使得序列升序的代价（j<m）,令dp[i][m]是不改变第i个数使得序列升序的代价。所求答案是min{dp[i-1][]}

对于每次状态转移需维护最小值，所以初始化所有状态为无穷大，再预先计算初始状态dp[0][]，即只有一个数的序列改变需要一次操作，不改变不需要操作，dp[0][j] = 1, 0<=j<m，dp[0][m] = 0。

对于状态的转移有四种情况。

在改变第i个时，需要一次操作次数。

  • dp[i][j]由dp[i-1][m]+1转移而来，条件是arr2[j] > arr1[m]
  • dp[i][j]由dp[i-1][c]+1, 0<=c<j转移而来，实际上是由dp[i][j-1]与dp[i-1][j-1]+1转移, (dp[i][j] = min{dp[i-1][0...j-1]+1} = min{dp[i-1][0...j-2]+1, dp[i-1][j-1]+1} = min{dp[i][j-1], dp[i-1][j-1]+1})

在不改变第i个时，不需要操作次数。

  • dp[i][m]由dp[i-1][m]转移而来，条件是arr1[i] > arr1[i-1]
  • dp[i][m]由dp[i-1][j]转移而来, 条件是arr1[i] > arr2[j]
